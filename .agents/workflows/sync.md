# NurKit — Sync

Save session state before closing or switching models.

## Steps

1. Read `.nurkit/prompts/sync.md` completely
2. Read the latest checklist
3. Overwrite the Current State section at the top of `.nurkit/CONTEXT.md`
4. Append a new session summary entry at the bottom — never delete old entries
5. Confirm: "Session saved. Safe to close. Next session continues at [exact task]."
