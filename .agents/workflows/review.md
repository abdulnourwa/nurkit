# NurKit — Review

Compare what was built against the blueprint and find drift.

## Steps

1. Read `.nurkit/prompts/review.md` completely
2. Read `.nurkit/CONTEXT.md`, latest blueprint, `.nurkit/STACK.md`, latest checklist
3. Review feature completeness, code quality, type integrity, security
4. Log all findings in `.nurkit/KNOWN-ISSUES.md`
5. Report summary to user with recommended actions

⚡ MODEL HINT: Use your most capable model for this step.
